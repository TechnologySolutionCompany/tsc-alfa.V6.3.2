⚠ Aviso Importante: Esta Política descreve como a TSC BR coleta, utiliza, compartilha e protege suas informações pessoais em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei 13.709/2018).

01 Âmbito e Definições

Esta Política aplica-se a todos os serviços oferecidos pela TSC, incluindo desenvolvimento de software, soluções em nuvem, consultoria em TI e serviços gerenciados.

  • Dados Pessoais: qualquer informação relacionada a pessoa natural identificada ou identificável.
  • Titular: pessoa natural a quem se referem os dados pessoais.
  • Controlador: TSC, que decide sobre o tratamento dos dados pessoais.
  • Operador: terceiro que processa dados em nome da TSC.
  • ANPD: Autoridade Nacional de Proteção de Dados.

02 Princípios de Tratamento

Nossas atividades seguem os princípios fundamentais da LGPD:

⚖ Legalidade
Sempre teremos base legal para o tratamento — consentimento, contrato, obrigação legal ou legítimo interesse.
🎯 Finalidade
Coletamos dados apenas para propósitos específicos, explícitos e legítimos.
🔬 Minimização
Limitamos a coleta apenas ao estritamente necessário para as finalidades declaradas.
✅ Exatidão
Mantemos os dados precisos, completos e atualizados quando necessário.

03 Categorias de Dados Coletados

Dados de Identificação

  • Nome completo, data de nascimento, CPF/CNPJ
  • Endereço físico e eletrônico
  • Números de telefone e identificadores eletrônicos

Dados Financeiros

  • Dados de cartão de crédito (armazenados de forma tokenizada)
  • Histórico de transações e faturas
  • Informações bancárias para transferências

Dados Técnicos

  • Endereços IP, logs de acesso, user agent
  • Cookies e identificadores de dispositivo
  • Dados de telemetria de sistemas

05 Compartilhamento de Dados

Provedores de Infraestrutura

AWS, Azure e Google Cloud para hospedagem de serviços, com cláusulas contratuais de proteção de dados.

Autoridades Regulatórias

Quando exigido por lei, ordem judicial válida ou requisição de autoridade competente.

Parceiros Estratégicos

Para entrega de soluções integradas, com acordos de confidencialidade devidamente firmados. Transferências internacionais seguem as cláusulas contratuais padrão da Comissão Europeia.

06 Medidas de Segurança

Implementamos um Sistema de Gestão de Segurança da Informação baseado na ISO 27001:

  • Criptografia: AES-256 para dados em repouso; TLS 1.3 para dados em trânsito.
  • Controle de Acesso: modelo de privilégio mínimo com autenticação multifator.
  • Resiliência: backups diários com replicação geográfica.
  • Monitoramento: SIEM 24/7 com detecção de anomalias.
  • Governança: políticas documentadas e treinamentos anuais obrigatórios.
🚨 Notificação de Incidentes: em caso de violação que represente risco aos titulares, notificaremos a ANPD e os afetados em até 72 horas, conforme o artigo 48 da LGPD.

07 Direitos dos Titulares

👁 Acesso
Solicitar confirmação e cópia dos dados pessoais em tratamento.
✏ Correção
Retificar dados incompletos, inexatos ou desatualizados.
🗑 Exclusão
Solicitar a eliminação de dados tratados com base no consentimento.
📦 Portabilidade
Receber os dados em formato estruturado e interoperável.
⛔ Oposição
Opor-se ao tratamento realizado com base em legítimo interesse.
🔔 Informação
Ser informado sobre entidades com as quais seus dados foram compartilhados.

Envie solicitações para techsocompany@gmail.com. Responderemos em até 15 dias úteis.

08 Retenção de Dados

  • Exigências legais (ex: documentos fiscais por 5 anos)
  • Ciclo de vida do contrato + período de prescrição aplicável
  • Finalidades comerciais legítimas devidamente documentadas

Ao atingir o prazo, os dados são anonimizados ou eliminados de forma segura (padrão NIST SP 800-88).
